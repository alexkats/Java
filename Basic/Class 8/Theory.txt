Generics
Полиморфизм - как можно писать универсальный код, один код используется с разными типами

Ad-hoc-методы - для каждого типа просто сделать свое поведение
Универсальный - одинаковое поведение для всех типов (основано на том, что у нас есть наследование)

Пример: оператор сложения
1 + 1
1.0 + 1.0
1.0 + 1

printf - перегрузка по всем параметрам, кроме первого + передаем явно информацию о типах
